# 📡 TradingPulse News MAX BOT

**TradingPulse News Bot** — это автоматизированный агрегатор новостей, который собирает материалы с информационных ресурсов через RSS, проверяет на дубликаты, фильтрует и обогащает контент с помощью YandexGPT, а затем публикует в канал мессенджера **МАКС**.

## 🎯 Основные возможности

- 🔄 Автоматическое получение новостей по RSS-лентам  
- 🧠 Анализ и обогащение текста с помощью YandexGPT  
- 🗃️ Интеллектуальное исключение дубликатов (по заголовку, дате и контенту)  
- 📤 Публикация отфильтрованных и обработанных постов в канал МАКС  
- ⏳ Периодическая проверка источников через планировщик задач  
- 📦 Масштабируемая архитектура для поддержки новых источников и мессенджеров

## 🛠️ Используемый стек

## 🚀 Установка и запуск

### Предварительные требования

- [Docker](https://www.docker.com/) (v24+ рекомендуется)
- [Git](https://git-scm.com/)

### 1. Клонирование репозитория

```bash
git clone https://github.com/kravecil/tradingpulse-news-bot.git
cd tradingpulse-news-bot
```

### 2. Настройка окружения

```bash
cp .env.example .env
```

Отредактируйте файл .env, указав реальные данные:
- BOT_TOKEN — токен бота в МАКС
- CHAT_ID — ID канала для публикации (напр., @channelname)
- YANDEX_API_KEY & YANDEX_FOLDER_ID — учётные данные Yandex Cloud для доступа к YandexGPT


### 3. Сборка и запуск

```bash
docker compose up -d --build
```

После запуска бот начнёт периодическую проверку RSS-ленты и публикацию уникальных, обработанных новостей.
